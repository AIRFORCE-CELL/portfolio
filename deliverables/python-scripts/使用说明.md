## 📊 Excel 批量合并

把文件夹里所有 `.xlsx` 合并成一个大表。

```bash
pip install openpyxl

# 合并当前目录所有 Excel
python excel_merge.py ./数据文件夹 -o 汇总.xlsx

# 指定工作表
python excel_merge.py ./数据 -o 汇总.xlsx -s "Sheet2"
```

第一行是表头，最后一列自动加上"来源文件"。

## 📝 文件批量重命名

```bash
# 先预览（安全，不会真的改名）
python file_rename.py ./图片 --prefix 旅行_

# 确认无误后加 --go 执行
python file_rename.py ./图片 --prefix 旅行_ --go

# 序号命名 001/002/003
python file_rename.py ./图片 --numbering --go

# 替换文件名中的文字
python file_rename.py ./图片 -p "旧文字" -r "新文字" --go
```

⚠️ 一定要**先不加 --go 预览**，确认后再执行！
