# 🤖 Discord 机器人 — 使用说明

## 第一步：创建 Discord 应用

1. 打开 https://discord.com/developers/applications
2. 点右上角 "New Application"，起个名字，创建
3. 左侧菜单点 "Bot" → "Add Bot"
4. 找到 "Privileged Gateway Intents"，三个开关**全打开**
5. 点 "Reset Token" → "Copy" — **复制保存，这就是你的 DISCORD_TOKEN**

## 第二步：邀请机器人进服务器

1. 左侧菜单点 "OAuth2" → "URL Generator"
2. Scopes 勾选 `bot`
3. Bot Permissions 勾选:
   - Send Messages / Read Messages
   - Manage Messages
   - Kick Members / Ban Members
   - Read Message History
4. 复制底部生成的 URL，浏览器打开，选择你的服务器，点"授权"

## 第三步：配置 & 启动

```bash
# 1. 安装 Python 依赖
pip install -r requirements.txt

# 2. 配置
cp .env.example .env
# 用记事本打开 .env，把 DISCORD_TOKEN 改成你刚才复制的 Token

# 3. 启动！
python -m bot
```

看到 `🤖 XXX 已上线` 就成功了！

## 第四步：在 Discord 里试试

在频道里输入:
- `!ping` — 测延迟
- `!poll "中午吃啥" 米饭 面条 饺子` — 创建投票
- `!stats` — 看服务器统计

---

## 🧠 开启 AI 对话（可选）

在 `.env` 里填上 `OPENAI_API_KEY`，指定 `AI_CHANNEL_IDS`，
机器人就会在那些频道里自动用 GPT 回复。

```env
OPENAI_API_KEY=sk-your-key-here
AI_CHANNEL_IDS=123456789,987654321
```
