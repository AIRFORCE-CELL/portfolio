# 🌐 个人主页 — 修改指南

## 文件说明

只有 **一个文件**：`index.html`，用记事本就能改。

## 改什么？

### 1. 改名字 & 标题

搜索 `全栈开发`，改成你的名字或领域：

```html
<title>你的名字 | AI 自动化 | 接单服务</title>
```

```html
<h1>AI 驱动的全栈开发</h1>
<!-- 改成 -->
<h1>张三 | 专业爬虫开发</h1>
```

### 2. 改服务卡片

搜索 `服务项目`，找到 6 个卡片，改服务名和价格：

```html
<h3>Discord / Telegram 机器人</h3>
<p>改成你的服务描述...</p>
<div class="price">¥800 - ¥3,000 <span>/ 起</span></div>
```

### 3. 改联系方式

搜索 `mailto:`，改成你的邮箱：

```html
<a href="mailto:你的邮箱@qq.com" class="cta-btn">📩 发送邮件</a>
```

### 4. 改技术栈

搜索 `技术栈`，改成你会的技术。

### 5. 改颜色

找到 `:root` 这行下面：
```css
--accent: #89b4fa;    /* 主色调，改成你喜欢的颜色 */
```

## 部署（免费）

改完后把 `index.html` 上传到 GitHub Pages，永久在线：

1. 打开 https://github.com/你的用户名/你的仓库/settings/pages
2. Source 选 "Deploy from a branch"
3. Branch 选 `main`，文件夹选 `/ (root)`
4. 点 Save，等 1 分钟就能访问了
