# 🕷️ 网页采集脚本 — 使用说明

## 安装

```bash
pip install beautifulsoup4
```

## 基础用法

```bash
# 采集网页所有 h1 标题
python scraper.py https://example.com

# 用 CSS 选择器采集特定内容
python scraper.py https://example.com -s ".article-title"

# 提取链接
python scraper.py https://example.com -s "a" -a href

# 结果保存到文件
python scraper.py https://example.com -s "h2" -o result.json
```

## 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `url` | 目标网址 | `https://xxx.com` |
| `-s` | CSS 选择器 | `-s ".price"` |
| `-a` | 提取属性 | `text` / `html` / `href` / `src` |
| `-o` | 输出文件 | `-o data.json` |

## 常用场景

```bash
# 电商价格监控
python scraper.py "https://item.jd.com/xxx.html" -s ".price" -o price.json

# 新闻标题抓取
python scraper.py "https://news.example.com" -s "h2.title" -o news.json

# 图片链接提取
python scraper.py "https://example.com/gallery" -s "img" -a src -o images.json
```

## 进阶：定时执行（Windows）

创建 `每日采集.bat`：
```bat
python scraper.py https://xxx.com -s ".data" -o today.json
```

然后用 Windows 任务计划程序设为每天自动运行。
